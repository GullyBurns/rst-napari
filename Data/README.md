* `plugin_descriptions.csv`:
	Name, Summary and Descr for all current plugins, from Justin Kiggins.	Date: Oct 13, 2021.

* `plugin_annotations.csv`:
	Ontology terms found for plugins in `plugin_descriptions.csv`.
